# 因果ループ図エディタ

複雑な系の構造を可視化するための、スタンドアロン HTML/JavaScript 製作図ツールです。  
インストール不要・サーバー不要。ブラウザで HTML ファイルを開くだけで動作します。

---

## 概要

因果ループ図（Causal Loop Diagram）は、システム思考において要素間の因果関係と循環構造を図示する手法です。  
本ツールは作図の 3 つの段階を **タブ形式の 3 つのビュー** で順に進められるよう設計されています。

```
① 要素ビュー  →  ② 相関ビュー  →  ③ 連鎖ビュー
（要素を列挙）   （リンクを作成）   （ループを解析・整形）
```

---

## 使い方

### 起動方法

```
causal-loop-editor.html をブラウザで開く
```

Chrome / Edge / Firefox など、モダンブラウザであれば動作します。

---

### ① 要素ビュー

システムを構成する要素を **1 行 1 要素** でテキスト入力します。

| 操作 | 方法 |
|------|------|
| テキスト編集 | 通常のテキストエディタと同様に操作 |
| 元に戻す | `Ctrl+Z` またはボタン |
| コピー | `Ctrl+C` またはボタン（選択範囲 or 全体） |
| 貼り付け | `Ctrl+V` またはボタン |
| 行削除 | 対象行にカーソルを置いてボタン |
| 行の入れ替え | 対象行にカーソルを置いて ↑ / ↓ ボタン |

**行の固定について**  
「② 相関」タブに初めて移動した時点で、入力済みの行数が**固定**されます。  
固定行は行番号に 🔒 マークが表示されます。  
固定後に途中へ行を挿入・削除しようとすると警告ダイアログが表示され、接続をリセットするか変更を元に戻すか選択できます。  
新しい要素は**末尾への追加**のみ可能です。

**接続情報パネル**（画面下部）  
相関ビューで結んだリンクの一覧が表示されます。「📋 コピー」でテキストとしてコピーできます。

---

### ② 相関ビュー

要素をノードとして配置し、因果関係をリンクで結びます。

#### ノード操作

| 操作 | 方法 |
|------|------|
| 移動 | ノード内をドラッグ |
| リサイズ | ノード右下の▪をドラッグ |
| 自動配置 | ツールバーの「⟳ 自動配置」ボタン（円形レイアウト） |

ノードを移動すると、接続済みのリンクの接続ポイントが**自動的に最適な位置に付け替え**られます。

#### リンク操作

各ノードには **8 つの接続ポイント**（上左・上中・上右・左・右・下左・下中・下右）があります。

| 操作 | 方法 |
|------|------|
| リンク作成 | ノード端の ● をドラッグして別のノードの ● に繋ぐ |
| リンク選択 | リンク（矢印）をクリック |
| 端点の付け替え | リンク選択後、矢印端の ● をドラッグして別の接続ポイントへ |
| リンク削除 | リンクを選択 → ポップアップの「✕」 |
| 反転マークの付与 | リンクを選択 → ポップアップの「⊿○」 |
| 遅延マークの付与 | リンクを選択 → ポップアップの「⊣⊢」 |

**反転（NOT）マーク**：因果が逆方向に働くことを示します（三角形＋小円）  
**遅延（コンデンサ）マーク**：因果の影響が時間遅れで現れることを示します（二重縦線）

#### 画像コピー

「🖼 画像コピー」ボタンでキャンバスを PNG としてクリップボードにコピーします。

---

### ③ 連鎖ビュー

相関ビューで作成したグラフを元に、閉じた因果ループを自動検出・表示します。

#### 自動解析

「🔍 自動解析」ボタンを押すと、リンクで閉じているループを全て検出します。

| ループ種別 | 条件 | 色 |
|-----------|------|-----|
| **強化ループ（R）** | 反転リンクの数が偶数（または 0） | 暖色系（赤・橙） |
| **均衡ループ（B）** | 反転リンクの数が奇数 | 寒色系（青・緑） |

各ループは**内接円弧の矢印**として表示されます。

#### ループの編集

ループ円をクリックすると編集パネルが開きます。

| 機能 | 説明 |
|------|------|
| ノード一覧 | ループを構成する要素の順序を表示 |
| ハイライト | 関連するノードとリンクが光芒でハイライトされる |
| サイズ（半径） | スライダーで円弧の大きさを調整 |
| 太さ | スライダーで円弧の太さを調整 |
| ドラッグ移動 | ループ円をドラッグして位置を変更 |
| 👻 薄く表示 | ループを半透明に（ループが多い場合の整理に） |
| ⊙ 内接円 | 算出された元の位置に戻す |

#### テキスト注釈

「✏ テキスト追加」ボタンでキャンバス上に自由なテキストボックスを追加できます。

| 操作 | 方法 |
|------|------|
| 追加 | ツールバーボタンからテキストを入力 |
| 移動 | テキストボックスをドラッグ |
| 編集 | テキストをクリック → ポップアップの「✎」 |
| 改行 | 編集中に `Shift+Enter` |
| 削除 | テキストをクリック → ポップアップの「✕」 |

#### 画像コピー

「🖼 画像コピー」ボタンで相関・連鎖レイヤーを合成した PNG をコピーします。

---

### 保存・ロード

作業内容は **JSON ファイル**として保存・読み込みができます。

| 操作 | 方法 |
|------|------|
| 保存 | 「💾 保存」→ ファイル名を入力 → `.json` ダウンロード |
| ロード | 「📂 ロード」→ 保存した `.json` ファイルを選択 |

保存されるデータ：テキスト・ノード位置とサイズ・リンク（反転/遅延を含む）・ループ設定・テキスト注釈

---

### 全てクリア

「🗑 全てクリア」ボタンを押すと確認ダイアログが表示されます。

- **💾 保存してから消去**：先に保存ダイアログを開いてから全データを消去
- **🗑 消去**：確認なしで全データを消去
- **キャンセル**：操作を中止

## 技術仕様

| 項目 | 内容 |
|------|------|
| 実装 | HTML5 / CSS3 / JavaScript（フレームワーク不使用） |
| 描画 | Canvas 2D API |
| ファイル形式 | 単一 HTML ファイル（`causal-loop-editor.html`） |
| 外部依存 | なし |
| 動作環境 | Chrome / Edge / Firefox / Safari（モダンブラウザ） |
| 保存形式 | JSON（version: 2） |

### ループ検出アルゴリズム

有向グラフに対する深さ優先探索（DFS）ベースの単純サイクル列挙を使用しています。  
各サイクルについて、含まれる反転リンクの数の偶奇を判定してループ種別を決定します。
